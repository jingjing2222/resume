Open Source

react-native-nitro-geolocation

  • 유지보수가 둔화된 @react-native-community/geolocation을 React Native New Architecture/Nitro Modules 기반으로 재구현하고, iOS/Android native location API를 JSI direct callback 구조로 연결 
  • Modern API: getCurrentPosition, requestPermission, useWatchPosition 중심의 Promise/Hook API로 callback/watch ID 관리보다 현대적인 DX 제공 
  • /compat subpath로 기존 callback API의 drop-in replacement를 제공해 migration 비용을 낮춤 
  • @react-native-nitro-geolocation/rozenite-plugin을 함께 개발해 Rozenite DevTools에서 interactive map, preset, keyboard/manual input 기반 위치 mocking과 조정 지원  

@facebook/react-native

  • Android에서 overrideBySynchronousMountPropsAtMountingAndroid 기본값을 활성화해 Native Animated direct manipulation props가 stale Fabric mount props에 덮이던 레거시 기본 정책 수정 (repro )
  • StyleSheet.absoluteFillObject 제거 changelog 정정 기여

@toss/granite React Native Framework

  • @granite-js/vitest 신규 패키지 개발 : Vitest에서 React Native Testing Library를 사용할 수 있도록 Jest bridge, RN runtime bootstrap, native module mock, Flow stripping/source transform, mirror cache, 문서와 테스트 구현
  • @granite-js/vitest 안정화: consumer RN source 기반 polyfill 로딩 , Ubuntu 20 호환 Flow stripping , flat React Native transform cache와 manifest resolver 개선
  • @granite-js/plugin-rozenite 신규 패키지 개발 : Rozenite middleware를 Granite 플러그인으로 연결하는 패키지 구조, 빌드 설정, 타입, 공개 API 구성 및 auto-discovery 복구
